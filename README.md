## 🔧 Tips for best results

- Start with short prompts
- Add one instruction at a time
- Say what you want the output to look like
- Use plain words
- Test different prompt patterns
- Save prompts that work well

Example:
- Weak prompt: Write about AI
- Better prompt: Write a short, clear paragraph about AI for a beginner, using simple words and one example
